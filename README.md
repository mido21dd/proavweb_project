Description:

This project is an order management system designed to organize and track orders in an easy and efficient way.
It allows users to add new orders, view, edit, and delete order lists, while saving data in the database.
It also provides a user-friendly interface, designed to facilitate order tracking and progress within the organization.

How to run:
